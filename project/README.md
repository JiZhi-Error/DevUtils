# About

该目录主要存储可运行 Android 项目

## 目录结构

```
- project                                  | 根目录
   - Accessibility                      | Android 无障碍使用 ( Activity 栈 )
   - AppInfo                            | APP 信息提取器
   - Demo                               | 临时测试代码、库调用调试 Demo
   - GTPush                             | 个推推送 ( 逻辑 ) 处理 ( 含设备管理 )
   - JPush                              | 极光推送 ( 逻辑 ) 处理 ( 含设备管理 )
   - UMShare                            | 友盟分享 ( 逻辑 ) 处理
```
